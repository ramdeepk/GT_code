Jacobe 7.2.7, release date: 20-6-2005

  • fixed a problem with rules --opencommentspace=0 (51) and --commentstarspace=0 (159); these settings enabled * / to be contracted to */ resulting in illegal Java code. Example:

    resulted in 
    /**/ */
    The fix is to insert one space in such a case.

Jacobe 7.1.0, release date: 10-6-2004

  • Incorporated new Java 1.5.0 language features:
    • Enhanced for loop (JSR 201)
    • Generics (JSR 14)
    • Metadata (JSR 175)

      Currently, Jacobe does not fully support the metadata functionallity as described in JSR 175. The specification allows constant expressions as so-called MemberValues, but Jacobe only accepts Literals (JLS 15.8.1), ExpressionNames (JLS 6.5) and Class Literals (JLS 15.8.2).

    • Static Import (JSR 201)
    • Typesafe Enums (JSR 201)
    • Varargs (JSR 201)
